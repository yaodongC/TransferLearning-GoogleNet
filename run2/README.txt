This file contains three programs.
************************************

For run 2, run BOW_DENSE.m

************************************
Put training images in training folder.
Put test images in testing folder.
MATLAB 2017b with neural network 
toolbox is required.
VLFEAT libraries is required.


create_vocab.m is a function to create 
vocabularies by K-means clustering.


create_histogram.m is the function to 
compute the histogram of each image.


natsortfiles.m and natsort.m is used to 
sort files in the natural order.
